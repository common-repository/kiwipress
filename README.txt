Finally there is an IRC plugin for WordPress that works!
KiwiIRC is a time-tested, web-based IRC client which can be added to any website with a few clicks.
To use this plugin:
1. simply download and extract it into your plugins folder
2. configure your settings in the WordPress dashboard
3. then drop the short tag [kiwipress] into your page or post.
Instantly your users will be able to stay connected via IRC.

Links:
KiwiIRC - <a href="https://kiwiirc.com">https://kiwiirc.com</a>
KiwiPress Plugin - <a href="https://kiwipress.tk/latest.zip">https://kiwipress.tk/latest.zip</a>
Help? <a href="mailto:scott@londontrustmedia.com">scott@londontrustmedia.com</a>

== Installation ==
1. Upload the "kiwipress" folder to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Place "[kiwipress]" shortcode in your pages or posts.
